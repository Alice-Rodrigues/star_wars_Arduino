# star_wars_Arduino

Criando um projetinho de Robotica para tocar a musica do Star Wars.

Lista de Material:

1 Microcontrolador (Arduino)

1 Mini protoboard.(Uma protoboard comum também serve)

1 Cristal (Led)

1 Fonte de energia

1 Buzzer/Piezo

Fios (Muitos)

Jumpers(Muitod)
